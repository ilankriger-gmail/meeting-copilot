🔊 Configurar BlackHole (macOS)

BlackHole captura todo o áudio do sistema (Zoom, Teams, Spotify, etc). Uma vez configurado, o app detecta automaticamente.

1Instale o BlackHole 2ch: brew install blackhole-2ch
2Abra Configuração de Áudio MIDI (busque no Spotlight: "Audio MIDI Setup")
3Clique em + (canto inferior esquerdo) → Criar Dispositivo Multi-Saída
4Marque seu alto-falante padrão + BlackHole 2ch
5Clique com botão direito no Multi-Saída → "Usar este dispositivo para saída de som"
6Pronto! Marque "Áudio do sistema" no modal e o app detecta o BlackHole automaticamente

Sem BlackHole, o app tenta capturar via compartilhamento de tela (só funciona para abas do Chrome).
